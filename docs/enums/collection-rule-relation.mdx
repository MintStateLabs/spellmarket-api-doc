Specifies the relationship between the column and the condition.

```graphql
enum CollectionRuleRelation {
  CONTAINS
  ENDS_WITH
  EQUALS
  GREATER_THAN
  IS_NOT_SET
  IS_SET
  LESS_THAN
  NOT_CONTAINS
  NOT_EQUALS
  STARTS_WITH
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>CONTAINS</b></code>](#) 
> The attribute contains the condition.
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>ENDS_WITH</b></code>](#)

> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>EQUALS</b></code>](#) 
> The attribute is equal to the condition.

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>GREATER_THAN</b></code>](#)

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>IS_NOT_SET</b></code>](#) 
> The attribute is not set (equal to null).
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>IS_SET</b></code>](#) 
> The attribute is set (not equal to null).
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>LESS_THAN</b></code>](#) 
> The attribute is less than the condition.
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>NOT_CONTAINS</b></code>](#) 
> The attribute does not contain the condition.
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>NOT_EQUALS</b></code>](#) 
> The attribute does not equal the condition.
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionRuleRelation.<b>STARTS_WITH</b></code>](#) 
> The attribute starts with the condition.
