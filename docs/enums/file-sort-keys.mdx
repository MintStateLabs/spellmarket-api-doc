The set of valid sort keys for the File query.

```graphql
enum FileSortKeys {
  FILENAME
  ORIGINAL_UPLOAD_SIZE
  CREATED_AT
  ID
  RELEVANCE
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>FileSortKeys.<b>FILENAME</b></code>](#) 
> Sort by the filename value.
> 

#### [<code style={{ fontWeight: 'normal' }}>FileSortKeys.<b>ORIGINAL_UPLOAD_SIZE</b></code>](#) 
> Sort by the original_upload_size value.
> 

#### [<code style={{ fontWeight: 'normal' }}>FileSortKeys.<b>CREATED_AT</b></code>](#) 
> Sort by the created_at value.
> 

#### [<code style={{ fontWeight: 'normal' }}>FileSortKeys.<b>ID</b></code>](#) 
> Sort by the id value.
> 

#### [<code style={{ fontWeight: 'normal' }}>FileSortKeys.<b>RELEVANCE</b></code>](#) 
> Sort by relevance to the search terms when the query parameter is specified on the connection.
Don't use this sort key when no search query is specified
