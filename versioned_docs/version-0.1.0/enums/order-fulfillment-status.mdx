Represents the order's aggregated fulfillment status for display purposes.

```graphql
enum OrderFulfillmentStatus {
  FULFILLED
  IN_PROGRESS
  PARTIALLY_FULFILLED
  UNFULFILLED
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>OrderFulfillmentStatus.<b>FULFILLED</b></code>](#) 
> Displayed as Fulfilled. All the items in the order have been fulfilled.
> 

#### [<code style={{ fontWeight: 'normal' }}>OrderFulfillmentStatus.<b>IN_PROGRESS</b></code>](#) 
> Displayed as In progress. Some of the items in the order have been fulfilled, or a request for fulfillment has been
sent to the fulfillment service.
> 

#### [<code style={{ fontWeight: 'normal' }}>OrderFulfillmentStatus.<b>PARTIALLY_FULFILLED</b></code>](#) 
> Displayed as Partially fulfilled. Some of the items in the order have been fulfilled.
> 

#### [<code style={{ fontWeight: 'normal' }}>OrderFulfillmentStatus.<b>UNFULFILLED</b></code>](#) 
> Displayed as Unfulfilled. None of the items in the order have been fulfilled.
>
