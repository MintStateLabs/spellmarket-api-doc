The set of valid sort keys for the ProductMedia query.

```graphql
enum ProductMediaSortKeys {
  ID
  POSITION
  RELEVANCE
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>ProductMediaSortKeys.<b>ID</b></code>](#) 
> Sort by the id value.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductMediaSortKeys.<b>POSITION</b></code>](#) 
> Sort by the position value.
> 

#### [<code style={{ fontWeight: 'normal' }}>ProductMediaSortKeys.<b>RELEVANCE</b></code>](#) 
> Sort by relevance to the search terms when the query parameter is specified on the connection. Don't use this sort key when no search query is specified.
