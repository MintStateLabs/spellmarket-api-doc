The set of valid sort keys for the Event query.

```graphql
enum EventSortKeys {
  CREATED_AT
  ID
  RELEVANCE
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>EventSortKeys.<b>CREATED_AT</b></code>](#) 
> Sort by the created_at value.
> 

#### [<code style={{ fontWeight: 'normal' }}>EventSortKeys.<b>ID</b></code>](#) 
> Sort by the id value.
> 

#### [<code style={{ fontWeight: 'normal' }}>EventSortKeys.<b>RELEVANCE</b></code>](#) 
> Sort by relevance to the search terms when the query parameter is specified on the connection. Don't use this sort
key when no search query is specified.&#x007D;
>
