A publication is a group of products and collections that is published to sales channels.

```graphql
type Publication {
  project: Project!
  hasCollection(
  id: ID!
): Boolean!
  id: ID!
  name: String!
  supportsFuturePublishing: Boolean!
  collectionPublications(
  first: Int
  after: String
  last: Int
  before: String
  reverse: Boolean = false
): ResourcePublication!
  collections(
  first: Int
  after: String
  last: Int
  before: String
  reverse: Boolean = false
): ProductCollectionConnection!
  productPublications(
  first: Int
  after: String
  last: Int
  before: String
  reverse: Boolean = false
): ResourcePublicationConnection!
  products(
  first: Int
  after: String
  last: Int
  before: String
  reverse: Boolean = false
): ProductConnection!
}
```
