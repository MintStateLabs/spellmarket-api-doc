The possible severity levels for a resource alert.

```graphql
enum ResourceAlertSeverity {
  CRITICAL
  DEFAULT
  INFO
  SUCCESS
  WARNING
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>ResourceAlertSeverity.<b>CRITICAL</b></code>](#) 
> Indicates a critical alert. For example, a blocked project.
> 

#### [<code style={{ fontWeight: 'normal' }}>ResourceAlertSeverity.<b>DEFAULT</b></code>](#) 
> Indicates a neutral alert. For example, an accepted dispute.
> 

#### [<code style={{ fontWeight: 'normal' }}>ResourceAlertSeverity.<b>INFO</b></code>](#) 
> Indicates an informative alert. For example, an escalated dispute.
> 

#### [<code style={{ fontWeight: 'normal' }}>ResourceAlertSeverity.<b>SUCCESS</b></code>](#) 
> Indicates a success alert. For example, a winning a dispute.
> 

#### [<code style={{ fontWeight: 'normal' }}>ResourceAlertSeverity.<b>WARNING</b></code>](#) 
> Indicates an informative alert. For example, a new dispute.
>
