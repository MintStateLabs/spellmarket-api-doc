The set of valid sort keys for the Collection query.

```graphql
enum CollectionSortKeys {
  TITLE
  UPDATED_AT
  ID
  RELEVANCE
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>CollectionSortKeys.<b>TITLE</b></code>](#) 
> Sort by the title value.
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionSortKeys.<b>UPDATED_AT</b></code>](#) 
> Sort by the updated_at value.
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionSortKeys.<b>ID</b></code>](#) 
> Sort by the id value.
> 

#### [<code style={{ fontWeight: 'normal' }}>CollectionSortKeys.<b>RELEVANCE</b></code>](#) 
> Sort by relevance to the search terms when the query parameter is specified on the connection.
Don't use this sort key when no search query is specified.
