The possible content types for a file object.

```graphql
enum FileContentType {
  IMAGE
  FILE
  VIDEO
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>FileContentType.<b>IMAGE</b></code>](#) 
> A MSL hosted image.
> 

#### [<code style={{ fontWeight: 'normal' }}>FileContentType.<b>FILE</b></code>](#) 
> A MSL-hosted generic file.
> 

#### [<code style={{ fontWeight: 'normal' }}>FileContentType.<b>VIDEO</b></code>](#) 
> A MSL-hosted video file. It's recommended to use this type for all video files.
>
